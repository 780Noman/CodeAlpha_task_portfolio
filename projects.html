DocumentGPT

Developed an intelligent document Q&A application using Python and Streamlit, allowing users to have real-time conversations with their files (PDF, DOCX, TXT, CSV). The system uses LangChain to chunk text, which is then embedded with Hugging Face sentence-transformers and stored in a FAISS vector store for efficient semantic search. It implements a Retrieval-Augmented Generation (RAG) pipeline, feeding retrieved document context and user queries to Google's Gemini Pro model for accurate, context-aware answers. The application is delivered through a dynamic chat interface and includes a one-click document summarization feature.

Tech Stack:tack: Python, Streamlit, LangChain, Google Gemini, FAISS, Hugging Face Transformers, PyPDF2, python-docx, pandas
